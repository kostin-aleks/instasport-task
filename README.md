# Instasport

## Тестовое задание

## REST API для расписания тренировок.

Ядро: __Django, DRF__

База данных: __Sqlite__


Главная таблица хранит расписание по дням недели и времени дня.

Хранится начало и конец тренировки.

Тренировка по виду спорта привязана к спортзалу, который в составе спортклуба,
который расположен в городе и стране.

Тренировку проводит тренер (не обязательно).

***

Админ панель позволяет админу добавлять и изменять объекты.

__/admin__

***

Для доступа клиента есть два метода АПИ.

***

__/training__

без параметров

фильтрация через query_string

weekday - список дней недели через запятую без пробелов.

начинается с 1-воскресенье, дальше до 7-суббота

пример:  weekday=1  weekday=2,5


sport - список видов спорта, id через запятую без пробелов

например, sport=1  sport=1,3


фильтрация по времени начала тренировки

from_time - время начала равно или больше этого времени

to_time - время начала равно или меньше этого времени

время задаётся в формате HH:MM

***

для удобства добавлены два метода АПИ, выводящие виды спорта и дни недели

__/training/sport__ - возвращает список видов спорта с id и названием

***

__/training/weekdays__ - возвращает список дней недели

***

__/training/{id}__  - возвращает одну тренировку по id
